# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, constructive, and harassment-free.

## Our Standards

Positive behavior includes:

- being respectful of different viewpoints and experience levels;
- giving constructive, evidence-backed feedback;
- accepting responsibility for mistakes;
- focusing on what improves the project and its users.

Unacceptable behavior includes:

- harassment, insults, or personal attacks;
- publishing another person's private information without permission;
- disruptive conduct that makes collaboration unsafe or unproductive.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that do not align with this code of conduct.

Report concerns through the repository owner's GitHub profile or repository issues when appropriate.
